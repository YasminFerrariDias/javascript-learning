Fetch API
---------
Demonstra o uso da Fetch API para consumir uma REST API local com json-server.
Inclui requisições GET por id, GET geral e POST via formulário com async/await.

Para executar:
  npm install
  npm run server

Acesse: http://localhost:3333/products
Arquivos: index.html, main.js, server.json, package.json
