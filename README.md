# JavaScript Learning

Estudos práticos de JavaScript organizados por módulos, mostrando a evolução desde manipulação básica do DOM até conceitos mais modernos da linguagem.

---

## 📂 Estrutura

O projeto está dividido em três etapas principais:

### 1. Básico

Primeiros contatos com JavaScript no navegador:

* Manipulação de conteúdo e estilos
* Criação de elementos
* Eventos (click, input, formulário)
* Uso de query selectors

---

### 2. Intermediário

Trabalhando com estruturas e lógica:

* Arrays e objetos
* Estruturas de repetição
* Classes
* Datas e manipulação de texto

---

### 3. Antes de frameworks

Introdução a conceitos mais modernos:

* Métodos de array
* Imutabilidade
* Módulos
* Funções assíncronas
* Consumo de APIs
* Bundlers e compiladores

---

## 🎯 Objetivo

Organizar e consolidar os principais conceitos de JavaScript ao longo do aprendizado, com exemplos simples e diretos.

---

## ▶️ Como executar

Abra os arquivos `.html` no navegador para visualizar os exemplos.

---

## 💭 Observação

É possível perceber a evolução do código ao longo dos módulos, tanto na organização quanto na forma de estruturar as soluções.
